# Electrostatic Solvation Energy

This project is a MATLAB implementation for calculating the electrostatic component of solvation free energy.  
It focuses on a computational drug-design model that evaluates electrical energy based on charge positions and a surrounding sphere.

## Features

- Calculate a surface integral over a sphere
- Compute self-energy for each charge
- Calculate charge radius values
- Compute electrical energy
- Use energy output to evaluate connection quality
- Includes a project presentation explaining the mathematical model and workflow

## Requirements

- MATLAB or MATLAB Online

## Setup and Installation

Clone the repository:

```bash
git clone https://github.com/SharonMelanker/electrostatic-solvation-energy.git
cd electrostatic-solvation-energy
```

Open the MATLAB Live Script:

```text
DrugDesign.mlx
```

Run the project in MATLAB or MATLAB Online.
